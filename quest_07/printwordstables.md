# Quest 7.5 - PrintWordsTables

## Название файла Go
`printwordstables.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| *Write a function that receives a string slice and prints each element of the slice in a separate line.* | *Напишите функцию, которая принимает срез строк и выводит каждый элемент этого среза на отдельной строке.* |

---

## Пример

- **Ввод:**

```go
a := piscine.SplitWhiteSpaces("Hello how are you?")
piscine.PrintWordsTables(a)
```

- **Вывод:**

```
Hello
how
are
you?
```

---

## Решение

```go
package piscine

import "github.com/01-edu/z01"

// PrintWordsTables выводит каждый элемент среза строк на отдельной строке.
func PrintWordsTables(a []string) {
	for _, word := range a {        // внешний цикл перебирает слова
		for _, char := range word { // внутренний - символы слова
			z01.PrintRune(char)     // вывод каждого символа
		}
		z01.PrintRune('\n')         // новая строка после каждого слова
	}
}
```

---

## Ключевые понятия (концепции и инструменты)

| Элемент | Объяснение |
|--------|------------|
| **Срез строк (`[]string`)** | Динамический массив строк; функция получает список слов и выводит каждое из них. |
| **Тип `rune`** | 32-битовое значение, представляющее Unicode-символ; корректно обрабатывает многобайтовые символы. |
| **range по срезу** | `for _, word := range a` — итерация по элементам среза; `_` означает, что индекс нам не нужен. |
| **range по строке** | `for _, char := range word` — перебор символов строки (каждый символ — `rune`). |
| **z01.PrintRune** | Функция для вывода одного символа; используется в Piscine вместо `fmt`. |

---

## Теория: почему строки нужно печатать по рунам?

В Go строки — это UTF-8 последовательность байтов. Один Unicode-символ может занимать 1–4 байта.  
- `s[i]` работает с байтами и может ломать многобайтовые символы (кириллица, эмодзи, иероглифы).  
- `for _, r := range s` возвращает полные руны (символы) и это безопасно для любых языков.

Если в задаче встречается только ASCII оба способа подходят. Для надежности рекомендуется использовать руны.

### 1. Объявление функции

```go
package piscine

func PrintWordsTables(a []string) {
```
Функция принимает срез строк, который нужно вывести и ничего не возвращает а выводит на экран.

---

### 2. Перебор слова и символа слова - строим вложенный цикл

```go
	for _, word := range a {        
		for _, char := range word { 
			z01.PrintRune(char)     
		}
		z01.PrintRune('\n')        
	}
```

Берём по очереди каждое слово из среза `a`. Индекс нам не нужен.
Каждое слово перебираем по символам (`rune`) и печатаем по одному символу.  

Нам нужно вывести каждый элемент среза на новой строке, поэтому печатаем новую строку после того как операции во внутреннем цикле выполнятся.

---

## Примеры итераций (как работает)

Для входа `[]string{"Hi", "мир"}` печать идёт так:

- Внешний цикл: `word = "Hi"`  
  - внутренний: печатаем 'H', 'i'  
  - печатаем '\n'
- Внешний цикл: `word = "мир"`  
  - внутренний: печатаем 'м', 'и', 'р' (правильно, потому что руны)  
  - печатаем '\n'

Итог в консоли:
```
Hi
мир
```
