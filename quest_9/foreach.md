# Квест IX. ForEach

## Название файла (go)

```
foreach.go
```

---

## Что нужно сделать — Instruction

### Оригинал инструкции

Write a function ForEach that, for an int slice, applies a function on each element of that slice.

### Перевод

Написать функцию `ForEach`, которая принимает:

* функцию `f`, применяемую к элементу типа `int`;
* срез `[]int`;

и выполняет функцию `f` для каждого элемента этого среза.

---

## Нам нужно написать функцию

Функция должна перебрать все элементы среза и вызвать переданную функцию для каждого из них.
Она **не возвращает значение**, а выполняет действия, определённые внутри `f`.

Пример использования:

```go
ForEach(func(x int) {
    fmt.Println(x)
}, []int{1, 2, 3})

// Вывод:
// 1
// 2
// 3
```

---

## Решение

```go
package piscine

// ForEach вызывает функцию action для каждого элемента из списка numbers.
// Пример: ForEach(print, []int{1,2,3}) выведет 1,2,3.
func ForEach(action func(int), numbers []int) {
    for _, value := range numbers { // перебираем элементы среза
        action(value)               // вызываем переданную функцию для каждого
    }
}
```

## ИТОГИ

### 1. Объявление функции

```go
func ForEach(action func(int), numbers []int)
```

Мы объявляем функцию, которая принимает:

* функцию `action`, принимающую одно число;
* срез `numbers`, содержащий элементы типа `int`.

Функция ничего не возвращает, потому что её задача — просто *выполнить действия* над каждым элементом.

---

### 2. Перебор элементов среза

```go
for _, value := range numbers {
```

`range` позволяет пройти по всем элементам среза.

* `_` — индекс (не нужен).
* `value` — текущее число.

---

### 3. Применение функции к каждому элементу

```go
action(value)
```

Функция `action` вызывается столько раз, сколько элементов в `numbers`.

## Заключение

Функция `ForEach` — важный базовый инструмент функционального подхода в Go. Она позволяет абстрагироваться от логики обхода и передавать внутрь действия, которые должны выполняться над каждым элементом.

Понимание таких функций помогает легко работать с коллекциями, обрабатывать данные и строить чистый и читаемый код.
