# Квест IX. AdvancedSortWordArr

## Название файла (go)

```
advancedsortwordarr.go
```

---

## Что нужно сделать — Instruction

### Оригинал инструкции

Write a function `AdvancedSortWordArr` that sorts a slice of strings, based on the function `f` passed as parameter.

### Перевод

Написать функцию `AdvancedSortWordArr`, которая принимает:

* срез строк `[]string`;
* функцию сравнения `f(a, b string) int`.

Функция должна сортировать срез **по правилу, заданному функцией `f`**:

* `f(a, b)` возвращает `1`, если `a` должен быть после `b`;
* `f(a, b)` возвращает `0`, если порядок неважен;
* `f(a, b)` возвращает `-1`, если `a` должен быть перед `b`.

Сортировка выполняется на месте (модифицирует исходный срез).

Пример использования:

```go
compare := func(a, b string) int {
    if len(a) > len(b) {
        return 1
    } else if len(a) < len(b) {
        return -1
    }
    return 0
}

words := []string{"apple", "banana", "fig", "kiwi"}
AdvancedSortWordArr(words, compare)
// words будет: ["fig", "kiwi", "apple", "banana"]
```

---

## Решение

```go
package piscine

// AdvancedSortWordArr сортирует срез слов words по правилу, заданному функцией f
func AdvancedSortWordArr(words []string, f func(a, b string) int) {
	for i := 0; i < len(words)-1; i++ {
		for j := i + 1; j < len(words); j++ {
			if f(words[i], words[j]) == 1 { // если первый элемент должен быть после второго
				words[i], words[j] = words[j], words[i] // меняем местами
			}
		}
	}
}
```

## ИТОГИ

### 1. Объявление функции

```go
func AdvancedSortWordArr(words []string, f func(a, b string) int)
```

Функция:

* принимает срез строк и функцию сравнения;
* сортирует строки по правилу `f`;
* модифицирует исходный срез, не возвращает новый.

---

### 2. Логика сортировки

1. Перебираем все пары элементов среза.
2. Вызываем функцию сравнения `f`.
3. Если функция возвращает `1`, меняем элементы местами.
4. В конце срез отсортирован по заданному правилу.

---

### 3. Особенности

* Сортировка **гибкая**: можно задавать любую функцию сравнения (`по длине, по алфавиту, по ASCII, по количеству гласных и т.д.`).
* Сортировка выполняется **на месте** — не создаётся новый срез.
* Метод с двойным циклом подходит для небольших массивов или учебных задач.

## Заключение

Функция `AdvancedSortWordArr` — универсальный инструмент сортировки строк в Go. Она позволяет реализовать **любую пользовательскую логику сортировки**, что делает её мощным инструментом для обработки коллекций строк.
