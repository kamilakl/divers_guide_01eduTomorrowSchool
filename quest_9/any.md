# Квест IX. Any

## Название файла (go)

```
any.go
```

---

## Что нужно сделать — Instruction

### Оригинал инструкции

Write a function `Any` that returns `true`, for a string slice:

if, when that string slice is passed through an `f` function, at least one element returns `true`.

### Перевод

Написать функцию `Any`, которая принимает:

* функцию `f`, принимающую `string` и возвращающую `bool`;
* срез `[]string`.

Функция должна возвращать `true`, если хотя бы один элемент среза при применении функции `f` возвращает `true`. Иначе — `false`.

Пример использования:

```go
result := Any(func(s string) bool {
    return len(s) > 3
}, []string{"hi", "hello", "go"})

// result будет: true, так как "hello" имеет длину больше 3
```

---

## Решение

```go
package piscine

// Any возвращает true, если хотя бы один элемент из среза a
// при применении функции predicate возвращает true.
func Any(predicate func(string) bool, a []string) bool {
	for _, value := range a {         // перебираем все элементы среза
		if predicate(value) {        // проверяем условие через функцию
			return true             // если элемент подходит, сразу возвращаем true
		}
	}
	return false                     // если ни один элемент не подошёл, возвращаем false
}
```

## ИТОГИ

### 1. Объявление функции

```go
func Any(predicate func(string) bool, a []string) bool
```

Функция принимает:

* функцию `predicate`, проверяющую условие на каждом элементе;
* срез строк `a`;

и возвращает `true`, если хотя бы один элемент удовлетворяет условию.

---

### 2. Перебор элементов среза

```go
for _, value := range a {
```

Используем `range`, чтобы пройти по всем элементам среза.

* `_` — индекс, нам не нужен.
* `value` — текущий элемент.

---

### 3. Проверка условия и возврат результата

```go
if predicate(value) { return true }
return false
```

* Если хотя бы один элемент прошёл проверку — возвращаем `true`.
* Если ни один — возвращаем `false` после завершения цикла.

## Заключение

Функция `Any` позволяет быстро проверить, удовлетворяет ли хотя бы один элемент коллекции заданному условию. Она полезна для фильтрации данных, поиска соответствий и написания чистого, читаемого кода.
