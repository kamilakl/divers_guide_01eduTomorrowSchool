# Квест IX. SortWordArr

## Название файла (go)

```
sortwordarr.go
```

---

## Что нужно сделать — Instruction

### Оригинал инструкции

Write a function `SortWordArr` that sorts by ASCII (in ascending order) a string slice.

### Перевод

Написать функцию `SortWordArr`, которая принимает срез строк `[]string` и сортирует его **по ASCII-кодам символов в порядке возрастания**.

Пример использования:

```go
words := []string{"banana", "apple", "cherry"}
SortWordArr(words)
// words будет: ["apple", "banana", "cherry"]
```

---

## Решение

```go
package piscine

// SortWordArr сортирует срез строк words по возрастанию ASCII
func SortWordArr(words []string) {
	for i := 0; i < len(words)-1; i++ {
		for j := i + 1; j < len(words); j++ {
			if words[i] > words[j] {      // сравниваем строки по ASCII
				words[i], words[j] = words[j], words[i] // меняем местами, если порядок нарушен
			}
		}
	}
}
```

## ИТОГИ

### 1. Объявление функции

```go
func SortWordArr(words []string)
```

Функция:

* принимает срез строк `words`;
* сортирует строки **по возрастанию ASCII**;
* не возвращает значения, так как сортирует срез на месте.

---

### 2. Логика сортировки

* Используется **простая сортировка пузырьком** (или похожий метод с двойным циклом).
* Сравниваем строки напрямую с помощью `>` — Go сравнивает строки посимвольно по ASCII.
* Если порядок нарушен — меняем элементы местами.

---

### 3. Особенности

* Сортировка **чувствительна к регистру** (`"A"` < `"a"` по ASCII).
* Подходит для небольших срезов строк.
* Работает «на месте» — не создаёт новый срез.
